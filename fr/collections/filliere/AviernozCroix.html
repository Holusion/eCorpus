Description

Cette ancienne croix de procession est remontée sur une nouvelle base tripode (à trois pieds). Elle possède des motifs à claire-voie qui représentent trois larges volutes. La croix et la base sont séparées par une hampe décorée de feuilles. Cette hampe se gonfle d’un nœud au centre. Un Christ de petite dimension est fixé sur la croix avec le visage levé vers le ciel. Le pied est habillé de décors de fleurons.
