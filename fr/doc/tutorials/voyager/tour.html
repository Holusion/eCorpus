Voyager Story

Editeur de scènes pour Voyager

Apprenez à créer des visites guidées en utilisant l'éditeur Voyager Story, développé par le Smithsonian Institute.

Créer une visite guidée

Créer une visite

Dans l'onglet {% include button.html name="visites" icon="globe" %}, cliquez sur le bouton {% include button.html icon="plus" %} pour créer une nouvelle visite guidée.

Vous pouvez alors lui donner un titre et un résumé, qui seront affichés lors de la selection d'une visite dans le menu pricipal.

Attention à bien parametrer la visite dans la langue de votre application.

Options de visite

Vous pouvez choisir quels paramètres pourront être modifiés d'une slide à une autre dans vos visites guidées.

Pour celà, allez dans les options, et cocher ceux désirés.

Ajouter une nouvelle slide

Dans le Tour Editor, cliquez sur Create pour créer une nouvelle étape pour votre visite guidée, puis définissez lui un titre.

Animer l'étape

Selon les options définies précédement, vous pouvez directement animer la camera dans la scène, ainsi que l'affichage d'articles et annotations.

Quand l'affichage vous plait, cliquez sur le bouton {% include button.html name="Update" icon="camera" %} pour enregistrer votre étape.

Animer les paramètres

Selon les options choisies, vous pouvez aussi animer certains paramètres. Pour celà, tout en gardant le Tour Editor ouvert sur la bonne visite, aller dans l'onglet {% include button.html name="Paramètres" icon="eye" %}.

Navigez dans votre scène et modifiez tous les paramètres souhaités. Le ou les modèle(s), les lumières, la scène, la caméra... Sans oublier de cliquer sur {% include button.html name="Update" icon="camera" %} pour enregistrer l'étape losque vous avez fini de la paramétrer.

Organiser les étapes

Une fois vos étapes créées, vous pouvez les réorganiser dans le Tour Editor en changeant leur ordre grace aux boutons {% include button.html name="Up" %} et {% include button.html name="Down"%}.

Vous pouvez aussi définir la durée et la courbe d'animation de la transition entre 2 étapes.

Vous avez à présent toutes les clefs en main pour créer et modifier des visites guidées.
