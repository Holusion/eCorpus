Title:

Mercure Temple

High school:

Lycée de l'Acheuléen - Amiens (France)

Material:

Stone

Description

This work was inspired by a column found in the Mercure Temple, located in Ribemont-sur-Ancre. It testifies the roman inspiration found in Gaul, France.

This column was replicated by hand, using traditionnaly seen pattern from the Temple's era. Polychromy was added to enhance this masterpiece.
